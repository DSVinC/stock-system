## 📊 7个核心绩效指标

1. **总收益率 (totalReturn)** - (期末值 - 期初值) / 期初值
2. **年化收益率 (annualizedReturn)** - (1 + 总收益率)^(365/天数) - 1
3. **夏普比率 (sharpeRatio)** - (超额收益 / 波动率) × √252
4. **最大回撤 (maxDrawdown)** - max((峰值 - 谷值) / 峰值)
5. **波动率 (volatility)** - 日收益率标准差 × √252
6. **胜率 (winRate)** - 盈利交易次数 / 总交易次数
7. **交易次数 (tradeCount)** - 完成的交易总数

# TASK_OPT_006: 条件单设计优化 - 因子权重条件面板

## 任务描述
设计并实现因子权重条件面板，类似音效调节面板的可视化交互界面。

## 需求来源
主人 2026-03-20 指示：
> "应该有个因子权重条件面板：所有带因子权重计算的指标都要给出当前权重方案和调节功能（类似音效调节面板。）"

## 功能要求

### 1. 因子权重面板
- [ ] 可视化滑块/旋钮界面（类似音效均衡器）
- [ ] 实时显示当前权重值
- [ ] 支持拖拽调节权重
- [ ] 权重总和自动校验（可选归一化）
- [ ] 预设方案快速切换

### 2. 支持的因子类型
- [ ] 技术指标因子（MA, MACD, KDJ, RSI 等）
- [ ] 基本面因子（PE, PB, ROE 等）
- [ ] 资金面因子（成交量，资金流向等）
- [ ] 情绪面因子（涨跌幅，波动率等）

### 3. 权重方案管理
- [ ] 保存自定义权重方案
- [ ] 加载历史方案
- [ ] 方案对比功能
- [ ] 方案回测验证

### 4. 实时反馈
- [ ] 权重变化实时影响条件单触发
- [ ] 可视化权重分布图
- [ ] 因子贡献度分析

## 技术实现
- 前端：HTML5 + CSS3 + JavaScript ( sliders/knobs UI )
- 后端：Node.js API 存储权重配置
- 数据库：weight_schemes 表

## 验收标准
1. 界面美观，操作流畅
2. 权重调节实时生效
3. 方案可保存/加载
4. 与现有条件单系统无缝集成

## 优先级
🟡 重要（回测系统前置依赖）

## 预计工时
8-12 小时

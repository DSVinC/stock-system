## 🎨 页面功能

### 1. 四维度行业权重
- 政策支持（滑块 0-100%）
- 商业化显现 ⭐（滑块 0-100%）
- 舆论热度 ⭐（滑块 0-100%）
- 资本认可（滑块 0-100%）
- 实时显示权重总和（绿色=100%，红色=异常）

### 2. 筛选阈值
- 营收增速下限（%）
- 毛利率下限（%）
- 热度前 N（%）
- 7 因子最低分（0-1）
- PE 上限
- PEG 上限

### 3. 核心 - 卫星仓位
- 核心仓比例（滑块 50-90%）
- 卫星仓比例（滑块 10-50%）
- 卫星仓股票数（滑块 1-5）
- 实时显示仓位总和

### 4. 网格交易参数
- 网格步长（%）
- 价格区间（下拉：3 月/6 月/12 月）
- 单笔金额（元）
- 趋势过滤（启用/禁用）

### 5. 风控参数
- 最大回撤（%）
- 最小年化收益（%）
- 最小胜率（%）

### 6. 配置管理
- 加载默认配置
- 保存配置
- 重置配置
- 配置列表
- 设为默认
- 删除配置

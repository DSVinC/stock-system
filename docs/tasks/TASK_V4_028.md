## 📐 技术方案

**数据源**: BaoStock（免费）

**回填脚本逻辑**:
```javascript
// 1. 登录 BaoStock
const bs = await baostock.login();

// 2. Top10 核心股票池
const stocks = ['sh600519', 'sz000858', ...];

// 3. 逐只股票回填分钟线
for (const stock of stocks) {
  const data = await bs.query_history_k_data_plus(
    stock,
    "date,time,open,high,low,close,volume",
    "2022-01-01",
    "2026-03-24",
    "5"  // 5 分钟线
  );
  await saveToDatabase(data);
}

// 4. 验证数据完整性
const count = await db.query('SELECT COUNT(*) FROM stock_minute WHERE trade_date >= "2022-01-01"');
console.log(`2022-2026 数据量：${count}`);
```

### 目标结构
在分析报告 JSON 中增加 `strategy` 字段：

```json
{
  "buyZone": [10.50, 11.20],
  "stopLoss": 9.80,
  "targetPrice": 13.50,
  "strategy": {
    "firstBuyPercent": 0.05,
    "maxPosition": 0.15,
    "addOnConditions": [
      { "type": "price", "value": 12.00, "percent": 0.05 },
      { "type": "breakout", "condition": "放量突破布林上轨", "percent": 0.05 }
    ],
    "riskProfile": "balanced"
  }
}
```

## 程序员任务内容
**分配给：Claude Code**

**工作内容：**

1. **修改个股分析技能** (`skills/a股个股分析/`)
   - 找到生成分析报告的代码位置
   - 在输出的JSON报告中增加 `strategy` 字段
   - 从分析报告中提取以下信息：
     - `firstBuyPercent`: 首次建仓比例（如 0.05 = 5%）
     - `maxPosition`: 最大仓位（如 0.15 = 15%）
     - `addOnConditions`: 加仓条件列表
     - `riskProfile`: 风险偏好（aggressive/balanced/conservative）

2. **策略数据来源**
   - 从分析报告的自然语言描述中提取结构化数据
   - 如果无法提取具体数值，设置合理的默认值
   - 保持与原始分析建议一致

3. **保留原有功能**
   - 不改变现有报告格式和内容
   - 只在JSON输出中增加新字段
   - 保证原有API兼容性

4. **测试验证**
   - 运行个股分析技能，生成至少3个不同股票的报告
   - 验证JSON输出包含正确的 `strategy` 字段
   - 验证数据提取的准确性

5. **文档更新**
   - 更新技能文档，说明新增的 `strategy` 字段
   - 提供示例用法

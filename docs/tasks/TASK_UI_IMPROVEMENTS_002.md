## 🎯 子任务

### TASK_UI_002 - 条件单导入分析报告流程优化
**当前问题**:
- 用户需要手动复制 JSON 报告 → 粘贴到条件单创建页面的文本框
- 体验差，容易出错，不知道有哪些报告可选

**设计方案**:
```
监控池选股（单选/多选）
    ↓
选择分析报告（从历史报告列表）
    ↓
选择投资风格（激进/稳健/保守）
    ↓
自动映射生成条件单
    ↓
确认并创建
```

**所需文件**:
- `conditional-order.html` - 新增导入向导 UI
- `api/analysis.js` - 获取历史报告列表 API
- `api/conditional-order.js` - 批量创建条件单 API

**预计工时**: 2-3 小时

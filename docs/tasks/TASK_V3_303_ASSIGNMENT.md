## 📋 任务描述

实现 Walk-Forward Analysis（前向分析）机制，验证优化结果在样本外数据上的表现，防止过拟合。

## 🎯 验收标准

### 核心功能
- [ ] 支持训练集/测试集时间分割（如 70%/30%）
- [ ] 在训练集上优化参数
- [ ] 在测试集上验证参数有效性
- [ ] 比较训练集 vs 测试集的绩效指标差异
- [ ] 检测过拟合（训练集/测试集差异 > 30% 则告警）

### 分割策略
- [ ] 支持固定比例分割（如 70%/30%）
- [ ] 支持滚动窗口（Rolling Window）
- [ ] 支持扩展窗口（Expanding Window）

### 输出要求
- [ ] 训练集绩效报告
- [ ] 测试集绩效报告
- [ ] 过拟合检测报告
- [ ] 参数稳定性分析

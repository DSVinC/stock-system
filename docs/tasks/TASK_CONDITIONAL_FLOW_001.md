# TASK_CONDITIONAL_FLOW_001 - 条件单股票选择器支持监控池

## 📋 实现详情

### 已实现功能
1. ✅ 股票选择器改为下拉选择（数据源：监控池）
2. ✅ 保留手动输入作为备选
3. ✅ 添加"手动输入"切换按钮
4. ✅ 选择监控池股票后自动填充代码和名称

- `conditional-order.html` - 添加股票选择器 UI 和逻辑

## ✅ 验收结果
1. ✅ 打开条件单创建表单，股票选择器显示监控池股票列表

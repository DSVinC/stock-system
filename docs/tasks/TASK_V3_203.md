## 🎯 验收标准

### 功能要求

- [ ] 支持多周期回测（1 分钟、5 分钟、15 分钟、30 分钟、60 分钟）
- [ ] 回测结果包含分钟线特有指标（如：日内交易次数、持仓时长（分钟））

### 代码改动
- [ ] 修改 `api/backtest-engine.js` - 添加分钟线数据加载
- [ ] 修改 `api/backtest-report.js` - 添加分钟线指标
- [ ] 新增 `api/backtest-minute.js` - 分钟线回测专用逻辑（可选）

### 测试要求
- [ ] 单元测试覆盖分钟线回测逻辑
- [ ] 端到端测试：运行一次完整的分钟线回测

1. **数据加载**: 从 stock_minute 表读取，按时间排序
2. **网格策略**: 价格每变动 X% 触发买卖
3. **时间处理**: 分钟线时间格式（YYYY-MM-DD HH:MM:SS）
4. **性能优化**: 分钟线数据量大，需要缓存优化

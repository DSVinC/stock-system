## 任务概述

优化股票系统用户体验，包括导航栏排序、账户管理功能增强、条件单导入流程优化、个股分析展示统一。

## 已完成任务

### ✅ 1. 导航栏排序调整
- **修改文件**: `index.html`
- **改动内容**: 
  - 顶部导航栏：账户管理移到回测系统后面
  - 主导航卡片：账户管理移到回测后面
- **新顺序**: 选股分析 → 个股分析 → 监控池 → 条件单 → 回测系统 → 账户管理

### ✅ 2. 账户管理 - 清空持仓功能
- **修改文件**: 
  - `api/portfolio.js` - 添加 `clearPositions` API
  - `api/server.js` - 添加路由 `POST /api/portfolio/account/:id/clear-positions`
  - `portfolio.html` - 添加清空持仓按钮和 `clearAllPositions` 函数
- **功能说明**:
  - 在持仓概览页面添加「清空持仓」按钮
  - 清空所有持仓并按当前价格生成卖出交易记录
  - 资金退回账户初始状态
  - 解决无法删除账户的问题（先清空持仓再删除）

### ✅ 3. 账户管理 - 删除账户功能
- **修改文件**: 
  - `api/portfolio.js` - 添加 `deleteAccount` API
  - `api/server.js` - 添加路由 `DELETE /api/portfolio/account/:id`
  - `portfolio.html` - 添加删除按钮和 `deleteAccount` 函数
- **功能说明**:
  - 每个账户卡片右侧添加删除按钮（🗑️）
  - 删除前检查：无持仓、无条件单、无交易记录
  - 保护数据安全，防止误删

### ✅ 4. 条件单 - 导入回测按钮
- **修改文件**: `conditional-order.html`
- **改动内容**:

  - 添加 `importToBacktest` 函数
  - 跳转到回测页面并传递条件单 ID
- **修复问题**: 回测系统引导中提到「导入回测」按钮但实际不存在

## 需求描述

条件单监控时间从当前配置优化为：9:30-11:30，13:00-15:00。

### 当前问题
- 监控时间配置不够精确
- 没有考虑午休时间
- 非交易日也可能运行

### 具体要求
1. **上午时段**: 9:30-11:30（每5分钟检查一次）
2. **午休**: 11:30-13:00（不监控）
3. **下午时段**: 13:00-15:00（每5分钟检查一次）
4. **交易日**: 仅工作日（周一至周五）
5. **停盘后**: 15:00后停止监控

## 技术细节

### 需要修改的文件
1. **定时任务配置**：
   - OpenClaw cron 任务：`stock-conditional-order-monitor`
   - 路径：通过 `cron list` 查看

2. **可能的脚本文件**：
   - `stock-system/scripts/conditional-order-monitor.mjs`
   - 可能需要添加时间判断逻辑

### cron表达式说明
- `0 */5 9-11,13-14 * * 1-5`
  - 分钟：0（每小时的0分）
  - 小时：9-11,13-14（9点到11点，13点到14点）
  - 天：任意
  - 月：任意
  - 星期：1-5（周一到周五）

注意：15:00不再监控，因为15:00是收盘时间，最后检查在14:55。

# TASK_V3_204: 网格参数优化模块

**优先级**: P1  
**工期**: 2 天  
**阶段**: Phase 2（网格交易回测）

---

## 📋 任务描述

实现网格交易参数优化模块，自动寻找最优网格步长、仓位比例等参数。

---

## 🎯 验收标准

### 功能要求
- [ ] 支持网格步长优化（范围 0.5%-2.0%，步长 0.1%）
- [ ] 支持仓位比例优化（范围 10%-50%，步长 5%）
- [ ] 支持多目标优化（收益率、夏普比率、最大回撤）
- [ ] 支持参数扫描（全组合回测）
- [ ] 输出最优参数组合

### 代码改动
- [ ] 新增 `api/grid-optimizer.js` - 网格参数优化器
- [ ] 新增 `scripts/run_grid_optimizer.mjs` - 命令行工具
- [ ] 修改 `api/optimizer.js` - 整合网格优化（可选）

### 性能要求
- [ ] 单次参数扫描耗时 < 5 分钟（100 个参数组合）
- [ ] 支持并行计算（4 核并行）

---

## 📁 交付物

- `api/grid-optimizer.js` - 网格参数优化器
- `scripts/run_grid_optimizer.mjs` - 命令行工具
- `test/grid-optimizer.test.js` - 单元测试

## 🔧 技术要点

1. **参数空间**: 步长、仓位、网格数量的组合
2. **优化算法**: 网格搜索、贝叶斯优化
3. **多目标**: 收益率↑、回撤↓、夏普↑
4. **可视化**: 参数热力图（可选）

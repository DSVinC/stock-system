## 📊 评分系统总览

### 6 因子评分模型

| 因子 | 权重 | 评分范围 | 核心作用 |
|------|------|----------|----------|
| 趋势因子 | 20% | 0.6 ~ 1.2 | 判断均线排列和价格趋势 |
| 动能因子 | 18% | 0.6 ~ 1.2 | 判断 MACD、RSI 动能状态 |
| 估值因子 | 18% | 0.6 ~ 1.2 | 判断 PE、PB 估值高低 |
| 资金因子 | 16% | 0.6 ~ 1.2 | 判断主力资金流向 |
| 业绩因子 | 16% | 0.6 ~ 1.2 | 判断营收、利润增速 |
| 波动率因子 | 12% | 0.6 ~ 1.2 | 判断 ATR、布林带波动 |

**综合评分映射**: 加权得分 → 1~5 分报告评级  
**决策建议**: ≥4.2 分=买入，≥3.2 分=观望，<3.2 分=回避

## 🔍 因子详细计算逻辑

### 1. 趋势因子 (Trend Factor) - 20%

**计算依据**: 均线排列、价格位置

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| 均线多头排列 | 25% | MA5>MA10 (+0.25), MA10>MA20 (+0.25), MA20>MA60 (+0.25), 价格>MA20 (+0.25) |
| 价格在 MA20 上方 | 30% | 是 (+0.3), 否 (0) |
| 价格在 MA60 上方 | 20% | 是 (+0.2), 否 (0) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**判断信号**:
- `maSignal`: 多头排列/空头排列/震荡
- 价格相对 MA20/MA60 位置

---

### 2. 动能因子 (Momentum Factor) - 18%

**计算依据**: MACD、RSI、MACD 柱状线

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| MACD 信号 | 40% | 金叉 (1.0), 多头延续 (0.8), 死叉 (0.2), 空头 (0.3), 中性 (0.5) |
| RSI(14) | 35% | >70 超买 (0.7), >60 偏强 (0.9), >50 中性 (0.7), >40 中性 (0.5), >30 偏弱 (0.3), <30 超卖 (0.2) |
| MACD 柱状线 | 25% | 红柱>0.5 (1.0), 红柱≤0.5 (0.8), 绿柱<-0.5 (0.2), 绿柱≥-0.5 (0.4) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**判断信号**:
- `macdSignal`: 金叉/死叉/多头延续/空头
- `rsiSignal`: 超买/超卖/中性

---

### 3. 估值因子 (Valuation Factor) - 18%

**计算依据**: PE(TTM)、PB、行业 PE 中位数、历史 PE 分位数

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| PE(TTM) 绝对值 | 40% | <15 (1.0), <25 (0.9), <35 (0.7), <50 (0.5), <80 (0.3), ≥80 (0.1) |
| 相对行业 PE | 30% | PE/行业 PE<0.6 (1.0), <0.8 (0.9), <1.0 (0.7), <1.2 (0.5), <1.5 (0.3), ≥1.5 (0.1) |
| 历史 PE 分位数 (5 年) | 20% | <10% (1.0), <20% (0.9), <30% (0.8), <50% (0.6), <70% (0.4), <90% (0.2), ≥90% (0.1) |
| PB | 10% | <2 (1.0), <4 (0.8), <6 (0.6), <10 (0.4), ≥10 (0.2) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**数据来源**:
- PE(TTM)、PB、总市值：Tushare `daily_basic` 表
- 行业 PE 中位数：Tushare `ths_index` 表
- 历史 PE 分位数：Tushare `stk_factor` 表

---

### 4. 资金因子 (Capital Factor) - 16%

**计算依据**: 主力资金净流入、主力/散户占比

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| 净流入额 (万元) | 60% | >5000 (1.0), >2000 (0.9), >1000 (0.8), >500 (0.7), >0 (0.6), >-500 (0.4), >-1000 (0.3), ≤-1000 (0.2) |
| 主力占比 | 40% | 主力/ (主力 + 散户)>0.3 (1.0), >0.2 (0.9), >0.1 (0.7), >0 (0.5), >-0.1 (0.3), ≤-0.1 (0.2) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**数据来源**:
- 同花顺主力净流入：Tushare `moneyflow_ths` 表
- 资金流向明细：Tushare `moneyflow` 表

### 5. 业绩因子 (Earnings Factor) - 16%

**计算依据**: 营收增速、净利润增速、ROE、毛利率

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| 营收同比增速 (%) | 30% | >50 (1.0), >30 (0.9), >20 (0.8), >10 (0.7), >0 (0.6), >-10 (0.4), ≤-10 (0.2) |
| 净利润同比增速 (%) | 35% | >100 (1.0), >50 (0.9), >30 (0.8), >20 (0.7), >10 (0.6), >0 (0.5), >-20 (0.3), ≤-20 (0.2) |
| ROE(%) | 20% | >20 (1.0), >15 (0.9), >12 (0.8), >10 (0.7), >8 (0.6), >5 (0.5), >0 (0.3), ≤0 (0.2) |
| 毛利率 (%) | 15% | >50 (1.0), >40 (0.9), >30 (0.8), >20 (0.6), >10 (0.4), ≤10 (0.2) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**数据来源**:
- 利润表：Tushare `income` 表
- 财务指标：Tushare `fina_indicator` 表

### 6. 波动率因子 (Volatility Factor) - 12%

**计算依据**: ATR、布林带宽度

| 子指标 | 权重 | 评分标准 |
|--------|------|----------|
| ATR/价格比值 | 60% | >8% (0.3), >5% (0.5), >3% (0.8), >2% (1.0), >1% (0.7), ≤1% (0.4) |
| 布林带宽度 | 40% | >20% (0.3), >15% (0.5), >10% (0.8), >5% (1.0), ≤5% (0.6) |

**原始得分范围**: 0 ~ 1.0  
**映射公式**: `score = 0.6 + rawScore × 0.6`

**数据来源**:
- ATR(20): 基于日线收盘价计算
- 布林带：MA20 ± 2×标准差

## 📐 综合评分计算

### 加权得分公式

```javascript
weightedScore = 
  trend.score × 0.20 +
  momentum.score × 0.18 +
  valuation.score × 0.18 +
  capital.score × 0.16 +
  earnings.score × 0.16 +
  volatility.score × 0.12
```

### 评级映射

| 综合得分 | 报告评级 (1-5 分) | 决策建议 |
|----------|------------------|----------|
| ≥4.6 | 5.0 | 买入 |
| ≥4.2 | 4.0 ~ 5.0 | 买入 |
| ≥3.2 | 3.0 ~ 4.0 | 观望 |
| <3.2 | 1.0 ~ 3.0 | 回避 |

## 🎯 建仓区间与止损计算

### 建仓区间

```javascript
// 技术支撑区间
techLow = min(MA60, 布林下轨)
techHigh = MA20

// 估值调整
if PE < 20:  // 低估值
  valueLow = 价格 × 0.95
  valueHigh = 价格 × 1.15
elif PE > 40:  // 高估值
  valueLow = 价格 × 0.85
  valueHigh = 价格 × 1.05
else:
  valueLow = 价格 × 0.9
  valueHigh = 价格 × 1.1

// 综合建仓区间
buyLow = max(techLow, valueLow)
buyHigh = min(techHigh, valueHigh)

// 限制在±15% 范围内
buyLow = max(buyLow, 价格 × 0.85)
buyHigh = min(buyHigh, 价格 × 1.15)
```

### 止损点

```javascript
if decision == '回避':
  stopLoss = 价格 × 0.92  // 回避时止损 8%
else:
  stopLoss = min(建仓下限 × 0.95, 价格 × 0.92)
```

### 目标价

```javascript
if decision == '回避':
  targetPrice = 价格 × 1.05
else:
  // 基于评分确定倍数
  if reportScore >= 4.5:
    multiplier = 1.25
  elif reportScore >= 3.5:
    multiplier = 1.15
  else:
    multiplier = 1.08
  
  targetPrice = max(建仓上限 × multiplier, 布林上轨)
```

## 📋 策略参数配置

### 风险偏好分级

| 风险类型 | 最大仓位 | 止损幅度 | 要求放量 |
|----------|----------|----------|----------|
| 激进型 | 20% | 8% | 是 |
| 稳健型 | 15% | 6% | 否 |
| 保守型 | 10% | 4% | 否 |

### 建仓策略

| 决策 | 风险类型 | 首次建仓 | 加仓条件 |
|------|----------|----------|----------|
| 买入 | 稳健型 | 5% | 突破建仓上限 (+5%), 放量突破布林上轨 (+5%) |
| 买入 | 激进型 | 5%~10% | 同上 |
| 观望 | 稳健型 | 2.5% | 站稳 MA20 且量比>1.5 |
| 观望 | 保守型 | 1.25% | MACD 金叉 + 基本面改善 |
| 回避 | 所有类型 | 0% | 不参与 |

## 📊 目标价格测算

### 时间维度目标价

| 时间 | 计算逻辑 | 预期涨幅 |
|------|----------|----------|
| 1 个月 | max(价格×1.08, MA10) | 4%~8% |
| 3 个月 | max(价格×1.16, MA20) | 9%~16% |
| 6 个月 | max(价格×1.28, 布林上轨) | 14%~28% |
| 乐观情景 | 价格×1.42 (买入) / 价格×1.18 (观望) | 18%~42% |

### 动态调整

- **买入决策**: 目标价上浮 8%~28%
- **观望决策**: 目标价上浮 4%~18%
- **回避决策**: 目标价下调 3%~8%

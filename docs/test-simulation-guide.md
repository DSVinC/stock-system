# 股票系统端到端模拟测试指南

## 测试角色设定

你是一名**量化投资新手用户**，有 100 万虚拟资金，想要使用这套股票系统进行：
1. 行业选股
2. 个股深度分析
3. 加入监控池
4. 创建条件单
5. 模拟交易
6. 回测验证

## 测试流程

### 阶段 1: 行业选股 (select.html)

**操作步骤**:
1. 打开 http://127.0.0.1:3000/select.html
2. 查看系统推荐的热门行业方向（AI、新能源、医药等）
3. 选择 2-3 个感兴趣的行业方向
4. 点击"开始选股"，查看系统筛选出的股票列表

**验证点**:
- [ ] 行业方向卡片正常显示
- [ ] 选股结果按评分排序（高分到低分）
- [ ] 每只股票显示：名称、代码、行业、市值、评分、决策建议
- [ ] 可以点击"查看报告"进入个股分析页面

---

### 阶段 2: 个股深度分析 (analysis.html)

**操作步骤**:
1. 从选股结果中选择 1 只股票（如：中际旭创 300308.SZ）
2. 点击"查看报告"进入个股分析页面
3. 查看完整的分析数据：
   - 核心技术指标（MA、MACD、RSI、布林带）
   - 估值分析（PE、PB、PEG）
   - 资金面数据（主力净流入、股东人数）
   - 多空观点对比
   - 三种风险偏好策略（激进/稳健/保守）
   - 建仓区间、止损点、目标价

**验证点**:
- [ ] 所有技术指标数据正常显示
- [ ] 估值数据与新浪财经一致
- [ ] 决策建议清晰（买入/观望/回避）
- [ ] 策略建议具体可执行
- [ ] 可以点击"导入条件单"跳转到条件单页面

**测试数据记录**:
```
股票名称：_______________
股票代码：_______________
当前价格：_______________
研究评级：___ / 5
决策建议：_______________
建仓区间：_____ ~ _____ 元
止损点：_____ 元
目标价：_____ 元
```

---

### 阶段 3: 加入监控池 (monitor-pool.html)

**操作步骤**:
1. 在个股分析页面，点击"加入监控池"按钮
2. 打开 http://127.0.0.1:3000/monitor-pool.html 查看监控池列表
3. 确认股票已成功添加

**验证点**:
- [ ] 添加成功提示显示
- [ ] 监控池列表刷新后显示新股票
- [ ] 显示添加时间、股票名称、代码
- [ ] 可以"查看报告"和"移除"操作

---

### 阶段 4: 创建条件单 (conditional-order.html)

**操作步骤**:
1. 从监控池选择 1 只股票，点击"批量创建条件单"
2. 或者在个股分析页面点击"导入条件单"
3. 配置条件单参数：
   - 交易动作：买入/卖出
   - 触发条件：股价上穿/下穿/高于/低于
   - 触发价格：根据分析建议设置
   - 仓位占比：5%~10%
   - 有效期：3 个月

**验证点**:
- [ ] 条件单表单正常显示
- [ ] 股票代码、名称自动填充
- [ ] 触发条件可选择（价格条件/技术指标/资金条件）
- [ ] 仓位占比、交易数量可设置
- [ ] 提交后显示"创建成功"
- [ ] 条件单列表中出现新订单

**测试数据记录**:
```
条件单 1:
- 股票：_______________
- 动作：买入/卖出
- 触发条件：股价_____ 元
- 仓位：_____%
- 状态：已启用/已提交

条件单 2:
- 股票：_______________
- 动作：买入/卖出
- 触发条件：股价_____ 元
- 仓位：_____%
- 状态：已启用/已提交
```

3. 查看账户总资产、可用资金、持仓市值

- [ ] 总资产 = 可用资金 + 持仓市值
- [ ] 可以查看持仓明细

### 阶段 6: 条件单监控与执行

**操作步骤**:
1. 等待条件单监控脚本运行（每 5 分钟检查一次）
2. 查看条件单状态变化：
   - 已提交 → 已启用 → 已触发 → 已执行
3. 检查账户持仓变化

**验证点**:
- [ ] 监控脚本正常运行（检查 cron 任务）
- [ ] 条件单状态正确更新
- [ ] 触发后执行买入/卖出操作
- [ ] 账户持仓、资金正确扣减/增加

**检查命令**:
```bash

tail -f /tmp/openclaw/*.log | grep conditional

# 查看数据库中条件单状态
sqlite3 /Users/vvc/.openclaw/workspace/stock-system/data/stock.db \
  "SELECT * FROM conditional_order LIMIT 5;"

# 查看持仓变化
sqlite3 /Users/vvc/.openclaw/workspace/stock-system/data/stock.db \
  "SELECT * FROM portfolio_position;"
```

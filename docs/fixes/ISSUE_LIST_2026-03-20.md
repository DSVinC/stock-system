### 问题 1: 首页标语冗余

**位置**: `index.html`  
**严重等级**: 低  
**问题描述**:  
首页顶部标语"基于新浪财经实时数据 + Tushare 历史数据的完整投资分析平台。支持选股、个股分析、监控、条件单、回测等功能。"过于冗长，不需要。

**解决方案**:
- 删除该段标语

**验收标准**:
- 首页不再显示该段标语
- 标题保持"A 股股票投资系统"即可

### 问题 2: 回测系统设计不符合需求

**位置**: `backtest.html`, `backtest.js`  
**严重等级**: 高  
**问题描述**:  
当前回测系统界面有"买入条件"和"卖出条件"输入框，设计不合理。需要改为策略选择 + 策略微调模式。

**解决方案**:
1. **去掉**：买入条件、卖出条件手动输入框
2. **增加**：策略选择下拉框（预设策略）
   - 双均线策略
   - RSI 超买超卖策略
   - MACD 金叉死叉策略
   - 布林带突破策略
   - 自定义条件单策略（从条件单导入）
3. **增加**：策略微调面板（根据选择的策略动态显示参数）
   - 均线周期（如 5 日/10 日/20 日）
   - RSI 阈值（如 30/70）
   - 其他策略参数
4. **改进**：股票代码输入方式
   - 改为搜索框，支持公司名/代码模糊搜索
   - 输入时实时检索，弹出选项
   - 勾选后添加到回测股票列表

**验收标准**:
- 用户可选择预设策略
- 策略参数可微调
- 股票搜索支持公司名和代码
- 支持多股票回测

### 问题 3: 选股页面 404

**位置**: `select.html` 缺失  
**严重等级**: 高  
**问题描述**:  
访问 `http://127.0.0.1:3000/select.html` 返回 404 错误：`{"success":false,"message":"未找到接口或页面：/select.html"}`

**解决方案**:
1. 创建 `select.html` 页面
2. 实现选股筛选功能（多维度筛选：PE、PB、ROE、技术指标、资金面、成长性）
3. 更新 `index.html` 中的链接，确保正确跳转

**验收标准**:
- `select.html` 可正常访问
- 支持多维度筛选
- 筛选结果可跳转到个股分析

### 问题 4: 配色风格不统一

**位置**: `portfolio.html`, `conditional-order.html`  
**严重等级**: 中  
**问题描述**:  
账户管理页面和条件单页面是浅色主题，与其他页面（首页/个股分析/监控池/回测系统）的深色主题不统一。

**解决方案**:
1. 统一所有页面为深色主题
2. 使用统一的 CSS 样式文件（`pages/style.css`）
3. 检查所有页面的配色方案：
   - 背景色：`#1a1a2e`（深蓝）
   - 卡片背景：`#16213e`
   - 文字颜色：`#eee`
   - 强调色：`#0f3460`, `#e94560`

**验收标准**:
- 所有页面风格统一
- 深色主题一致
- 无明显视觉割裂感

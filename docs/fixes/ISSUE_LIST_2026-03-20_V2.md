## 🔍 问题详情

### 问题 1: 导航栏样式不统一（高优先级）

**位置**: `portfolio.html`, `conditional-order.html`  
**严重等级**: 高  
**问题描述**:  
账户管理和条件单页面的导航栏位置和样式与其他页面存在差异，导致视觉割裂感。

**解决方案**:
1. 检查 `portfolio.html` 和 `conditional-order.html` 的导航栏结构
2. 统一为与其他页面一致的顶部导航栏设计
3. 使用统一的 CSS 样式文件（`pages/style.css`）

**验收标准**:
- 所有页面导航栏样式一致
- 无明显视觉割裂感

---

### 问题 2: 策略参数微调面板未显示（中优先级）

**位置**: `backtest.html`, `backtest.js`  
**严重等级**: 中  
**问题描述**:  
选择"双均线策略"后未见参数微调面板显示，功能未完全实现。

**解决方案**:
1. 检查 `onStrategyChange()` 函数实现
2. 确认 `params_double_ma` 等元素 ID 是否正确
3. 测试 `classList.add('active')` 是否正常生效

**验收标准**:
- 选择任意策略后，对应参数面板正常显示
- 面板包含正确的参数输入框

---

### 问题 3: 条件单导入功能界面不明确（中优先级）

**位置**: `backtest.html`, `conditional-order.html`  
**严重等级**: 中  
**问题描述**:  
在条件单页面未见明显的"导入"功能按钮，回测系统选择"自定义条件单策略"后未见具体导入界面。

**解决方案**:
1. 在条件单页面添加"导出/保存"功能
2. 在回测系统条件单策略面板中明确导入流程
3. 添加用户引导提示

**验收标准**:
- 条件单页面可导出/保存条件单
- 回测系统可导入已保存的条件单
- 用户流程清晰

---

### 问题 4: 空状态体验可优化（低优先级）

**位置**: `portfolio.html`  
**严重等级**: 低  
**问题描述**:  
账户管理页面显示"-"符号较多，空状态体验可优化。

**解决方案**:
1. 添加空状态引导提示
2. 添加"创建第一个账户"的快捷入口
3. 优化空数据展示样式

**验收标准**:
- 空状态有明确引导
- 用户体验友好

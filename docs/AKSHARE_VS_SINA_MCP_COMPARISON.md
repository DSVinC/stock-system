## 📦 数据字段对比

### Sina MCP 返回字段

```json
{
  "id": "string",
  "symbol": "sh603305",
  "name": "旭升集团",
  "title": "公告标题",
  "content": "公告内容摘要",
  "date": "2026-04-01",
  "time": "17:58:00",
  "timeStamp": 1712000000,
  "event_type": "regulatory_investigation",
  "market": "0",
  "scheme": "string",
  "attachment": "array"
}
```

### AkShare 返回字段

```json
{
  "关键词": "603305",
  "新闻标题": "公告标题",
  "新闻内容": "公告内容摘要",
  "发布时间": "2026-04-01 17:58:00",
  "文章来源": "财中社",
  "新闻链接": "http://..."
}
```

### 字段映射

| 系统需求字段 | Sina MCP | AkShare | 转换方式 |
|-------------|----------|---------|----------|
| `ts_code` | symbol | 关键词 | 直接映射 |
| `title` | title | 新闻标题 | 直接映射 |
| `content` | content | 新闻内容 | 直接映射 |
| `ann_date` | date | 发布时间.split(' ')[0] | 字符串分割 |
| `ann_time` | time | 发布时间.split(' ')[1] | 字符串分割 |
| `source` | event_type | 'akshare' | 固定值 |
| `url` | ❌ 无 | 新闻链接 | AkShare 独有 |
| `provider` | ❌ 无 | 文章来源 | AkShare 独有 |

**结论**: AkShare 字段完全满足系统需求，且提供额外信息

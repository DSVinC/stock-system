## 🔧 技术方案

### 9 个指标卡片

| 指标 | 字段 | 计算逻辑 |
|------|------|---------|
| 总收益率 | `total_return` | (期末值 - 期初值) / 期初值 |
| 年化收益 | `annualized_return` | 总收益年化 |
| 夏普比率 | `sharpe_ratio` | 超额收益 / 波动率 |
| 最大回撤 | `max_drawdown` | 最大峰值到谷值跌幅 |
| 胜率 | `win_rate` | 盈利次数 / 总次数 |
| 盈亏比 | `profit_loss_ratio` | 平均盈利 / 平均亏损 |
| 交易次数 | `total_trades` | 成交笔数 |
| 持仓周期 | `avg_holding_days` | 平均持仓天数 |
| Alpha | `alpha` | 超额收益 |

### 前端数据绑定修复

**修复前**:
```javascript
// 错误：直接访问嵌套对象
metrics.total_return
```

**修复后**:
```javascript
// 正确：检查数据结构
const metrics = result.data?.metrics || {};
const totalReturn = metrics.total_return ?? '--';
```

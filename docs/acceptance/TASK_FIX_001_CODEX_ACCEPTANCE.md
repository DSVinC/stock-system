## ❌ 发现的问题

### 问题 1: 导航栏样式不统一（高优先级）

**位置**: `portfolio.html`, `conditional-order.html`  
**问题描述**: 账户管理和条件单页面的导航栏位置和样式与其他页面存在差异，导致视觉割裂感。  
**影响**: UI 一致性受损，用户体验下降  
**修复建议**: 统一为与其他页面一致的顶部导航栏设计

### 问题 2: 策略参数微调面板未显示（中优先级）

**位置**: `backtest.html`, `backtest.js`  
**问题描述**: 选择"双均线策略"后未见参数微调面板显示。  
**影响**: 策略参数无法调整，功能不完整  
**修复建议**: 检查 `onStrategyChange()` 函数和 CSS `.active` 类

### 问题 3: 条件单导入功能界面不明确（中优先级）

**位置**: `backtest.html`, `conditional-order.html`  
**问题描述**: 条件单页面未见明显的"导入"功能按钮，回测系统选择"自定义条件单策略"后未见具体导入界面。  
**影响**: 用户无法完成条件单导入操作  
**修复建议**: 添加导出/导入功能，明确用户流程

### 问题 4: 空状态体验可优化（低优先级）

**位置**: `portfolio.html`  
**问题描述**: 账户管理页面显示"-"符号较多，空状态体验可优化。  
**影响**: 用户体验不佳  
**修复建议**: 添加空状态引导提示和快捷入口

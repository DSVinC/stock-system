# 策略库详细设计 (V5_009)

## 1. 双均线策略 (double_ma)

**参数**:
- `ma_short`: 短期均线周期 (5-30)
- `ma_long`: 长期均线周期 (30-200)
- `stop_loss`: 止损比例 (0.05-0.20)
- `take_profit`: 止盈比例 (0.10-0.50)

**信号逻辑**:
- 金叉：短期均线上穿长期均线 → 买入
- 死叉：短期均线下穿长期均线 → 卖出

---

## 2. RSI 超买超卖策略 (rsi)

**参数**:
- `rsi_period`: RSI 周期 (7-21)
- `overbought`: 超买阈值 (60-80)
- `oversold`: 超卖阈值 (20-40)

**信号逻辑**:
- RSI < oversold → 买入
- RSI > overbought → 卖出

---

## 3. MACD 金叉死叉策略 (macd)

**参数**:
- `fast_period`: 快线周期 (8-15)
- `slow_period`: 慢线周期 (20-30)
- `signal_period`: 信号线周期 (5-15)

**信号逻辑**:
- DIF 上穿 DEA → 金叉买入
- DIF 下穿 DEA → 死叉卖出

---

## 4. 布林带突破策略 (bollinger)

**参数**:
- `bb_period`: 布林带周期 (15-30)
- `bb_std`: 标准差倍数 (1.5-3.0)

**信号逻辑**:
- 价格跌破下轨 → 买入
- 价格突破上轨 → 卖出

---

## 5. 四维度七因子策略 (seven_factor)

**参数**:
- `trend_weight`: 趋势维度权重 (0.2-0.4)
- `momentum_weight`: 动量维度权重 (0.1-0.3)
- `value_weight`: 估值维度权重 (0.1-0.3)
- `moneyflow_weight`: 资金维度权重 (0.2-0.4)
- `pe_max`: PE 最大值 (20-50)
- `pb_max`: PB 最大值 (2-8)
- `roe_min`: ROE 最小值 (0.10-0.20)

**信号逻辑**:
- 四维度综合评分 > 阈值 → 买入
- 评分跌破阈值 → 卖出

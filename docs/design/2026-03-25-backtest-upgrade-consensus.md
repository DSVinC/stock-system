## 需求背景

### 当前问题
1. **选股 - 回测割裂**：选股结果无法直接用于回测验证
2. **未来函数风险**：回测使用当前参数而非选股时点参数
3. **策略配置不统一**：选股、回测、监控的策略参数无法共享
4. **牛熊市验证缺失**：缺少完整周期的回测验证

### 设计目标
1. **选股 - 回测联动**：回测界面内嵌选股，使用选股时点数据
2. **策略模板化**：支持多策略模板（4 维度 +7 因子、资金流、技术面等）
3. **参数双向流转**：选股↔回测策略参数可互相导入导出
4. **完整周期回测**：覆盖 2020-2026 牛熊周期验证

│     从 strategy_configs                   │     → strategy_configs

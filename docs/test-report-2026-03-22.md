## 🔧 已修复问题

### 问题 #1: PE 历史分位数容错
**问题描述**: 部分小盘股历史 PE 分位值获取失败导致评分偏差  
**修复方案**: 在 `score-factors.js` 中添加 null/undefined 检查，使用默认值 0.5  
**状态**: ✅ 已修复

### 问题 #2: 股票代码后缀自动补充
**问题描述**: 手动输入代码无后缀 (.SZ/.SH) 时后端校验报错  
**修复方案**: 在 `market-data.js` 的 `searchStock` 函数中自动识别并补充后缀：
- 6/9 开头 → .SH
- 3/0 开头 → .SZ
- 4/8 开头 → .BJ  
**状态**: ✅ 已修复

### 问题 #3: 飞书通知日志优化
**问题描述**: 未配置环境变量时打印警告但不中断  
**修复方案**: 优化日志输出为 info 级别，明确说明"跳过通知"而非"警告"  
**状态**: ✅ 已优化

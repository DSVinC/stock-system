## 🎯 优化目标

根据验收测试中发现的问题，优化以下三个场景的用户体验：

1. ✅ 行业选股页添加加载状态提示
2. ✅ 行业选股页添加数据为空提示

## 📝 优化内容

### 1. select.html - 行业选股页面

#### 新增空状态 UI 组件
```html
<div class="empty-state" id="emptyState" style="display: none;">
  <div class="empty-icon">📭</div>
  <h3 class="empty-title">暂无行业推荐数据</h3>
  <p class="empty-copy">当前没有可用的行业选股建议。可能是数据源暂未更新，请稍后重试。</p>
  <button class="refresh-btn" onclick="startSelection()" style="margin-top: 1rem;">重新选股</button>
</div>
```

#### 样式优化
- 新增 `.empty-state` 类：统一样式（居中、内边距、颜色）
- 新增 `.empty-icon` 类：大号图标，半透明效果
- 新增 `.empty-title` 类：醒目标题
- 新增 `.empty-copy` 类：柔和说明文字，限制最大宽度

#### 逻辑优化
```javascript
// 区分空数据和加载错误
if (error.message && error.message.includes('暂无行业推荐数据')) {
  emptyState.style.display = 'block';  // 显示空状态
  errorState.style.display = 'none';
} else {
  errorState.style.display = 'block';  // 显示错误状态
  emptyState.style.display = 'none';
  errorMessage.textContent = error.message;
}
```

**优化效果**:
- ✅ 用户看到友好的空状态提示，而不是空白页面
- ✅ 明确告知原因（数据源未更新）
- ✅ 提供"重新选股"操作按钮

### 2. monitor-pool.html - 监控池页面

#### 增强空状态样式
```css
.empty-state {

  margin-top: 1.5rem;
  display: flex;

  gap: 1rem;
}
```

#### 优化空状态内容
```html
<section id="emptyState" class="empty-state" aria-live="polite">
  <div class="empty-icon">📭</div>
  <h3 class="empty-title">监控池暂时为空</h3>
  <p class="empty-copy">
    当您在<strong>个股分析</strong>页面点击"加入监控池"后，股票会显示在这里。<br>
    监控池中的股票会被条件单监控系统跟踪，触发条件时自动执行。
  </p>
  <div class="empty-actions">
    <a href="./analysis.html" class="button-link button-primary">去添加股票</a>
    <button class="button-secondary" onclick="loadMonitorPool()">刷新列表</button>
  </div>
</section>
```

#### 表格显示逻辑优化
```javascript
function renderRows(stocks) {
  monitorTableBody.replaceChildren();

  if (!stocks.length) {
    emptyState.classList.add('visible');
    // 隐藏表格头部，只显示空状态
    document.querySelector('table').style.display = 'none';
    return;
  }

  emptyState.classList.remove('visible');
  // 显示表格
  document.querySelector('table').style.display = 'table';
  // ... 渲染行
}
```

**优化效果**:
- ✅ 空状态时隐藏表格框架，视觉更清爽
- ✅ 提供"去添加股票"快捷入口，引导用户操作
- ✅ 说明监控池的作用（条件单跟踪）
- ✅ 区分加载失败和真正空池的不同提示

## 🎨 设计原则

1. **视觉友好**: 使用 emoji 图标 (📭) 增加亲和力
2. **信息清晰**: 标题 + 说明两层结构，快速理解
3. **操作引导**: 提供明确的下一步操作按钮
4. **场景区分**: 区分空数据、加载错误、真正空池
5. **视觉简洁**: 空状态时隐藏无关 UI 元素

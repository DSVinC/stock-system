## 🔍 核心功能

### P0 任务功能
1. **黑天鹅检测**: 5 类风险关键词（39 个），一票否决机制
2. **舆情因子计算**: 新闻 + 公告整合，时间衰减，置信度加权
3. **因子快照库**: 按天归档，支持查询/统计/清理
4. **7 因子整合**: 权重配置，加权计算，综合评分 0.6-1.2

### P1 任务功能
1. **LLM 情感分析**: 智能情感分析，降级机制（关键词匹配）
2. **异步流水线**: 并发控制，进度追踪，错误处理
3. **时间衰减优化**: 差异化半衰期，多曲线类型，批量处理
